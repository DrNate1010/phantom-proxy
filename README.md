This is a Node JS application with npm packages. To run locally you must first run the following:
```bash
git clone https://github.com/DrNate1010/phantom-proxy
cd phantom-proxy
  ```
  <br>
  
```bash
  npm i
  pnpm i
  ```
  <br>

  ```bash
  pnpm start
  ```

You should have phantom loaded on localhost:8080!
